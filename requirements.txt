# Prerequisites for Running the Honeypot Project

## 1. Python Installed

Install **Python 3.10 or higher**

Download from:
[https://www.python.org/downloads/](https://www.python.org/downloads/)

After installation, check in terminal:

```bash
python --version
```

Expected output example:

```text
Python 3.11.5
```

---

# 2. Git Installed

Install Git from:

[https://git-scm.com](https://git-scm.com)

Check installation:

```bash
git --version
```

Example output:

```text
git version 2.52.0
```

---

# 3. Required Python Libraries

Install required dependencies using:

```bash
pip install -r requirements.txt
```

---

# 4. requirements.txt File

Create this file in the project root:

```text
Flask
```

Later modules may add:

```text
pandas
matplotlib
```

---

# 5. Clone the Repository

Each team member should run:

```bash
git clone <repository_link>
```

Example:

```bash
git clone https://github.com/yourusername/honeypot-login-monitor.git
```

---

# 6. Navigate to Project Folder

```bash
cd honeypot-login-monitor
```

---

# 7. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 8. Run the Honeypot Server

```bash
python app.py
```

---

# 9. Open in Browser

Open:

```text
http://127.0.0.1:5000
```

---

# Folder Structure After Cloning

```text
honeypot-project
│
├── app.py
├── attack_logs.csv
├── requirements.txt
│
├── templates
│   └── login.html
│
└── static
    └── style.css
```

---

# Important Note

If Python command does not work, try:

```bash
python3 app.py
```
